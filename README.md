# Strings

![Strings](screenshots/screenshot000.png "Strings")

## Description

Find a way to activate the goals with the right color.

## Controls

Keyboard/Mouse:
- WASD - Controls the character
- E - Activates Line Spawners
- C - Created new point/sets goal
- R - Reset
